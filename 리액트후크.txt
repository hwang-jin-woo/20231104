React의 기본 hook 4개

1. useState
  : 컴포넌트에서 값을 저장할 때 사용함
  : state가 변경되면 렌더링이 발생
2. useEffect
  : 관심 state의 값이 변했을 때에만 실행되는
  : 코드를 정의하는데 사용됨
  : []형태로 관심 state를 정의
  : 빈 배열은 최초1회만 useEffect가 실행된다는 의미
3. useRef
  : 컴포넌트에서 값을 저장할때 사용하는 점은 
  : useState와 동일하나 값이 변해도 렌더링 NO발생
4. useContext
  : 특정 props를 외부 메모리공간(=Context)에 저장하고
  : 어느 자식 컴포넌트에서건 엑세스가능하도록 함
